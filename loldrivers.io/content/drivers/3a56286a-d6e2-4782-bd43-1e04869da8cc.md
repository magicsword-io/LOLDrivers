# OAToolx64.sys ![:inline](/images/twitter_verified.png) 

### Description

OAToolx64.sys is a vulnerable kernel driver from the KeServiceDescriptorTable/vulnerable-drivers repository. The driver exposes dangerous kernel primitives to usermode.
- **UUID**: 3a56286a-d6e2-4782-bd43-1e04869da8cc
- **Created**: 2026-04-17
- **Author**: Michael Haag
- **Acknowledgement**:  | [@rainbowdynamix, @DbgPrint](https://twitter.com/@rainbowdynamix, @DbgPrint)
