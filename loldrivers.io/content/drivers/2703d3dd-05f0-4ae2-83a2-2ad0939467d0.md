# Cndom6.sys ![:inline](/images/twitter_verified.png) 

### Description

Signed malicious drivers reported in Silver Fox activity; rwdriver.sys exposes a rootkit IOCTL primitive, while Cndom6.sys and XiaoH.sys are reported as watchdog/support drivers.

### Commands

```
sc.exe create rwdriver binPath=C:\windows\temp\rwdriver.sys type=kernel &amp;&amp; sc.exe start rwdriver
```


| Use Case | Privileges | Operating System | 
|:---- | ---- | ---- |
| Load malicious signed kernel drivers used for rootkit and watchdog activity | kernel | Windows 10, Windows 11 |
