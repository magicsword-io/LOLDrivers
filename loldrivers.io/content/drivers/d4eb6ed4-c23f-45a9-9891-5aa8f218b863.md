# pcdsrvc_x64.sys ![:inline](/images/twitter_verified.png) 

### Description

PC-Doctor pcdsrvc_x64.sys is listed as a KDU provider for Dell PC Doctor / SupportAssist driver abuse. The driver family is associated with CVE-2019-12280 and exposes kernel access primitives used by KDU as a provider for kernel memory operations.
- **UUID**: d4eb6ed4-c23f-45a9-9891-5aa8f218b863
- **Created**: 2026-06-16
- **Author**: Michael Haag
