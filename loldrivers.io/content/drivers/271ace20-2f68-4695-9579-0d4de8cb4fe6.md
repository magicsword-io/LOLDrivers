# GameDriverX64.sys ![:inline](/images/twitter_verified.png) 

### Description

GameDriverX64.sys is a signed game driver reported in public DragonForce intrusion tradecraft and BlackSnufkin BYOVD research as an abused vulnerable driver. Public research identifies vulnerable builds that can be used as process-killer providers, including CVE-2025-61155 context.
