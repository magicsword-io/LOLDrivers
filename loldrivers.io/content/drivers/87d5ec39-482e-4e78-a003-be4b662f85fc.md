# vsdatant.sys ![:inline](/images/twitter_verified.png) 

### Description

Check Point ZoneAlarm driver (vsdatant.sys) abused in BYOVD attacks to gain kernel privileges and bypass protections such as Memory Integrity.
- **UUID**: 87d5ec39-482e-4e78-a003-be4b662f85fc
- **Created**: 2025-03-22
- **Author**: Kyaw Pyiyt Htet
- **Acknowledgement**: Venak Security | [](https://twitter.com/)

