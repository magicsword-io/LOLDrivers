# PCTcore64.sys ![:inline](/images/twitter_verified.png) 

### Description

PC Tools PCTcore64.sys is documented by BlackSnufkin BYOVD research and CERT/CC as CVE-2026-8501. The driver creates a PCTCore device interface without restrictive access controls and exposes a process-termination IOCTL reachable by local callers.
- **UUID**: 8f2d3be4-3e25-490f-82bc-0aca16aee101
- **Created**: 2026-06-16
- **Author**: Michael Haag
- **Acknowledgement**: BlackSnufkin | [@BlackSnufkin42](https://twitter.com/@BlackSnufkin42)
