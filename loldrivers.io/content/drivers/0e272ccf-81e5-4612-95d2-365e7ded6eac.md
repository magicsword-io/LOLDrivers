# shield.sys ![:inline](/images/twitter_verified.png) 

### Description

Horizon DataSys Shield drivers expose IOCTL functionality reported to provide arbitrary kernel read/write primitives.
- **UUID**: 0e272ccf-81e5-4612-95d2-365e7ded6eac
- **Created**: 2026-06-16
- **Author**: Michael Haag

### Commands

```
sc.exe create shield binPath=C:\windows\temp\shield.sys type=kernel &amp;&amp; sc.exe start shield
```


| Use Case | Privileges | Operating System | 
|:---- | ---- | ---- |
| Load vulnerable signed disk filter drivers for kernel memory access | kernel | Windows 10, Windows 11 |
